Introduction

Welcome to the Suhwa documentation. Suhwa provides state-of-the-art sign language recognition APIs and tools to help developers build inclusive applications.

Key Concept

Suhwa uses real-time computer vision and deep learning to translate sign language gestures into text or speech instantly.

Quickstart

Get started with our API in minutes. Here's a basic example using `curl` to analyze a video frame.

Bash 
curl -X POST https://api.suhwa.ai/v1/recognize \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "image=@frame.jpg" \
  -F "language=asl"

Recognition API

Our core API endpoint for real-time sign language recognition.

Parameter Type Description
image File The image frame to analyze (JPEG/PNG).
language String ISO code of the sign language (e.g., asl, isl).
threshold Float Confidence threshold (0.0 to 1.0). Default: 0.5.
