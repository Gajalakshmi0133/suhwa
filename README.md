🎨 💡 About SUHWA

🌟 **SUHWA** is a smart AI system that detects **Sign Language gestures** and converts them into readable text.

💬 Designed to **break communication barriers** and empower accessibility using cutting-edge **Machine Learning & Deep Learning**.

---

🚀 🔥 Features

✨ Real-time Gesture Detection  
📸 Image-based Prediction  
🧠 Deep Learning Models  
💬 NLP Text Processing  
🌐 Web-based Interface  
⚡ Fast & Scalable Backend
